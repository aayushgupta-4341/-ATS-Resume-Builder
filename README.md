# 📄 ATS Resume Builder

A simple and user-friendly web application to create ATS-friendly resumes.  
Users can fill in their details, preview the resume in real-time, and download it in PDF or DOCX format.

---

## Features

- Multiple Resume Templates (Classic & Modern)
- Profile Photo Upload with Preview
- Dynamic Form (Education, Experience, Projects)
- Field-based Resume (IT / Commerce / Arts / Other)
- Live Resume Preview
- Download Resume as PDF
- Download Resume as DOCX
- Support for LinkedIn, GitHub & Portfolio links
- Fully Responsive Design

---

## Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- html2pdf.js (for PDF generation)
- docx.js (for DOCX file generation)

## How It Works

1. Fill in your personal and professional details
2. Add sections like:
   - Education
   - Experience
   - Projects
3. Select a template (Classic / Modern)
4. Click on **Generate Resume**
5. Preview your resume on the right side
6. Download as:
   - PDF
   - DOCX

---

## Key Functionalities

- Dynamic form fields using JavaScript
